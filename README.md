# ChatHub
ChatHub is a multi-user android chat application.
#Features of ChatHub:
	•	Multi-user chat application
	•	Authentication
	•	Texting
	•	Photos Sharing
	•	Camera
	•	Cloud persistence using Firebase
	•	Music Files Sharing
	•	Draw and Send
	•	Location Sharing
#Some of the API’s and Libraries this application uses:

	•	Firebase Android API
	•	Google Map API
	•	Google + API
	•	MediaPlayer
	•	Camera API
	•	Espresso
	•	Recycler View
	•	Glide image library
